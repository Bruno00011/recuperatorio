Sitios Web y Servidores

Un sitio web es un conjunto de paginas a través de Internet y alojadas en un servidor.

Definiciones HTML y CSS

HTML (Hypertext Markup Language) y CSS (Cascading Style Sheets) son los lenguajes fundamentales para la creación y diseño de páginas web.

Estructura de Directorio

La estructura de directorio se refiere a la organización de archivos y carpetas en un proyecto web, lo que incluye la ubicación de los archivos HTML, CSS, imágenes y otros recursos.

Herramientas a utilizar

Las herramientas comunes utilizadas en desarrollo web incluyen editores de texto como Visual Studio Code. Navegadores web como Goggle Chrome para realizar pruebas y plataformas de control de versiones como Git Y Github para el trabajo colaborativo y el control de cambios en el código.

Etiquetas y Atributos

Las etiquetas se utilizan para estructurar el contenido de una página web, mientras que los atributos propocionan información adicional sobre las etiquetas. Por ejempolo, el átributo h r e f se utiliza en la etiqueta <a> para especificar la URL de destino de un enlace
